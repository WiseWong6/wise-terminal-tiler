Terminal Tiler

macOS 终端窗口平铺工具。一键整理散乱的 iTerm2、Terminal 和 Ghostty 终端窗口。为 Claude Code、Codex、OpenClaw 等多终端工作流设计。

GitHub 仓库

功能特性

⌨️

快捷键一键整理

默认 Ctrl+Cmd+T 即可将 2~10 个终端窗口自动排列成最优网格。

⊞

全屏 / 分区模式

支持全屏平铺和左侧 1/2、1/3、1/4 分区模式，方便搭配浏览器使用。

🖥️

多显示器支持

自动按显示器分组，每个屏幕独立平铺，互不干扰。

⌘

跨终端混用

支持 iTerm2、Terminal、Ghostty 三种终端同时平铺。

布局矩阵

根据窗口数量自动选择最优布局。

快速安装

在终端中运行以下命令，一键安装 Terminal Tiler 及其快捷键。

curl -fsSL https://raw.githubusercontent.com/WiseWong6/wise-labs/main/terminal-tiler/scripts/install-agent-commands | bash
